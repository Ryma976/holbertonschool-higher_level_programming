The `MyList` class inherits from the built-in `list` and adds a `print_sorted` method.

To use it, import the class:
    >>> MyList = __import__('1-my_list').MyList

Instantiate the class and append integers:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)

Printing the list displays elements in insertion order:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

The `print_sorted` method prints the elements sorted dynamically:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

The original list remains unmodified:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Testing with an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []
